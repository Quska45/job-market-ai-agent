# Job Market AI Agent

채용공고를 매일 수집하고, AI/개발 직무 요구사항을 분석해서, 질문 응답과 알림까지 제공하는 학습용 AI 에이전트 프로젝트입니다.

## 목표

이 프로젝트의 목표는 단순 챗봇이 아니라, 실제 데이터를 주기적으로 수집하고 분석하는 AI 시스템을 직접 만드는 것입니다.

최종적으로는 다음 기능을 갖습니다.

- 채용공고 데일리 수집
- 중복 공고 및 변경 공고 감지
- LLM 기반 공고 분석
- 기술스택, 경력, 직무 유형 분류
- 채용시장 트렌드 리포트 생성
- 사용자의 질문에 데이터 기반으로 답변
- 관심 조건에 맞는 공고와 변화 알림

## 핵심 학습 주제

- Python 백엔드 개발
- 웹 크롤링 및 데이터 정제
- 데이터베이스 설계
- LLM API 사용
- structured output
- embedding 및 RAG
- agent workflow
- 스케줄링
- 알림 자동화
- 테스트와 운영 품질

## MVP

첫 번째 MVP는 `AI 채용공고 데일리 브리핑`입니다.

기능 범위:

1. 한 개 채용 사이트에서 AI 관련 공고를 수집한다.
2. 공고를 정규화해서 저장한다.
3. LLM으로 직무, 경력, 기술스택을 분석한다.
4. 일일 요약 리포트를 생성한다.
5. CLI에서 질문하면 저장된 공고 기준으로 답변한다.

## 원칙

- 로그인 우회, 약관 위반, 과도한 요청은 하지 않는다.
- 가능한 공식 API, RSS, 공개 검색 결과, 정적 페이지부터 사용한다.
- 크롤링 대상 사이트별 robots.txt와 이용 조건을 확인한다.
- API 키와 비밀번호는 코드에 저장하지 않는다.
- LLM 결과는 검증 가능한 구조로 저장한다.
- 테스트 가능한 작은 단위로 만든다.
